# GMX 实现交互

1. 在 https://gmx.io/trade 开立杠杆交易
2. 在https://gmx.io/trade 上做交易 
3. 在 https://gmx.io/buy 网站上进行 Mint GLP

#  快速开始


## 安装相关包

```
npm install 
```

## 配置环境变量
 
```
cp .env.sample .env
```

## Swap ETH to USDC 

```
node swap.js
```

## Open and close ETH long

```
node leverageTrade.js
```
